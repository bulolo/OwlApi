# 简介

OwlApi 是一个企业级 **SQL to API** 智能网关平台，帮助开发者和企业快速构建及管理数据接口。

## 核心能力

- **极速发布** — 编写 SQL 查询，一键发布为标准 RESTful API 接口
- **多源连接** — 无缝连接 MySQL、PostgreSQL、Oracle 等各类主流数据库
- **混合云网关** — Runner 可运行在本地 IDC、云服务器或边缘设备，自动建立加密隧道
- **多租户隔离** — 支持多租户/多项目隔离管理，保障数据安全

## 技术栈

| 层级 | 技术 |
| :--- | :--- |
| Backend | Go 1.22, gRPC, Gin, GORM |
| Frontend | Next.js 14, React, TailwindCSS |
| Database | PostgreSQL, Redis |
| Infra | Docker, Docker Compose |
