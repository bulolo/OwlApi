# 更新日志

OwlApi 持续快速迭代。在这里，你可以了解每个版本的核心改动与新功能。

---

## v0.1.7 · 2026-05-13 🗄️ 多数据库全面接入

### 🔌 六类数据源支持
- 新增 MySQL、PostgreSQL、SQL Server、SQLite、StarRocks、Apache Doris 全面接入，驱动均已集成，开箱即用
- DSN 自动推断：优先使用 `db_type` 字段确定驱动，兜底使用 DSN 前缀识别，无需手动配置驱动类型

### 🖥️ 数据浏览器
- 新增表结构浏览（Schema 查看）功能，可视化展示远端数据库所有表与字段定义，无需额外工具
- 新增表数据预览，支持指定行数限制（最多 500 行）快速抽样查看

### 📋 结构化连接表单
- 数据源配置告别手写 DSN，改为可视化字段填写（主机、端口、用户名、密码、数据库名）
- 支持「测试连接」按钮，创建前实时验证连通性并返回延迟（ms）

### 🔧 其他改进
- 修复生产部署后网关/数据源无初始数据的问题
- 修复 `tenant` 列表 `ORDER BY` 导致超管登录默认租户错误的问题
- 修复登录成功后偶发卡在登录页的问题

## v0.1.6 · 2026-04-01 🌐 网关执行器演进

### 🛡️ 混合云架构
- 发布全新架构大版本：Gateway 可直接部署在深层内网，主动向控制面建立 gRPC 反向隧道
- 彻底告别公网 IP 和复杂的白名单配置，数据库防火墙端口无需对外开放
- 新增内置 Gateway：`docker-compose.yml` 自动包含本地网关节点，开箱即用

### ⚙️ 多租户 RBAC
- 上线 RBAC 体系（SuperAdmin / Admin / Viewer），实现多数据源、多项目的严格权限隔离
- SuperAdmin 可跨租户管理，Admin 管理租户内全部资源，Viewer 只读访问

### 📦 参数推断重构
- 支持在 SQL 编写时动态识别 `:变量`，一键映射并暴露为标准的 HTTP Query 参数或 Body
- 参数类型自动推断，支持手动配置 `required`、`default`、`desc` 等属性

### 📜 JS 脚本引擎
- 新增前置脚本（pre_script）和后置脚本（post_script）支持
- 基于 Goja 引擎执行，可在 SQL 执行前修改参数，在 SQL 执行后转换响应数据
- 支持独立脚本管理，可在多个接口间复用

### 🌐 版本管理与发布流
- 接口支持草稿（draft）→ 发布（published）工作流
- 发布时创建快照（Release），已发布接口调用快照执行，编辑草稿不影响线上
- 支持版本回滚和接口下线，租户可配置最大保留版本数

### 📚 官网全线升级
- [owlapi.cn](https://owlapi.cn) 官网焕新，全面适配企业级 UI 体验

## v0.1.5 · 2026-02-01 🏗️ 控制面核心完成

### 🎯 管理控制台上线
- Admin 控制台基于 Next.js 16 App Router 重构完成
- 实现租户管理、项目管理、接口 CRUD、数据源配置全链路
- 引入 hey-api TypeScript SDK 自动生成，确保前后端类型安全

### 🔐 认证系统
- JWT 认证体系完整实现，支持 Token 自动续期
- 新增 401 拦截器，Token 失效时自动跳转登录页
- 支持「记住账号」功能

### 📦 Docker 化
- 全栈 Docker Compose 方案完成，支持 `make dev-up` 一键启动六大服务
- 生产与开发环境配置分离，`make prod-init` / `make dev-init` 快速初始化
